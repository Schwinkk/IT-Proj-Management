# Отчет о реализации функционала «Экспорт оценок»

**Исполнитель:** [ФИО]  
**Задача:** Issue #1 — Добавить возможность экспорта списка оценок в CSV.

## 1. Выбор стратегии ветвления
Была выбрана стратегия **GitHub Flow**, так как она идеально подходит для веб-проектов с непрерывной доставкой (CI/CD).
*   Ветка `main` содержит всегда рабочий код.
*   Разработка велась в ветке `feature/csv-export-functionality`.
*   Слияние выполнено через механизм Pull Request (код-ревью).

## 3. Техническая реализация
Реализована функция на **Vanilla JS**, которая:
1.  Парсит DOM-дерево HTML-таблицы.
2.  Собирает массив данных.
3.  Добавляет `\uFEFF` (BOM) для поддержки кириллицы в Windows.
4.  Генерирует Blob-объект и инициирует скачивание.

## 4. Результат
Функционал успешно внедрен в `main`. Кнопка работает во всех современных браузерах.
